# Blood Bank Management System

This is a web application built with Flask for managing a blood bank system. It allows administrators to manage donor requests, track active donors, receive fund donations, and collect feedback from users.

## Features

- **Admin Dashboard**: Provides a comprehensive overview of donor requests, active donors, and fund donations.
- **Donor Management**: Admins can approve or reject donor requests and manage active donors.
- **User Interface**: Users can search for active donors based on blood group and location.
- **Fund Donations**: Users can make fund donations through card payment or UPI.
- **Feedback System**: Users can provide feedback about their experience.
